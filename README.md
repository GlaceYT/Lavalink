## 🚀 Getting Started

Follow these simple steps to get your Lavalink server up and running on any VPS or local machine.

### 1. Download Lavalink
Download the latest pre-configured zip file:
- **Direct Download**: https://github.com/GlaceYT/Lavalink/releases/tag/v4_MARCH_2_2026

### 2. Refresh OAuth Token
To ensure YouTube playback works without issues, you should refresh your OAuth token regularly:
- **Token Link**: [Refresh OAuth Token Here](https://glaceyt.com/resource/lavalinks)

### 3. Installation & Run
1. **Extract** the downloaded `.zip` file.
2. **Edit** your configuration if needed in `application.yml`.
3. **Start** the server using the following command:
   ```bash
   java -jar Lavalink.jar
   ```
